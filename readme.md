# My Record List
<img src="public\pics\my-record-list-github-banner.png" alt="My Record List" />

## Description
My Record List is a terminal application that has been created to help me experiment with senior level ways of code and system design. It's very simple in terms of functionality where it has Create, Read, Update and Delete features for vinyl records and artists using a mock database.

## Technology Stack
- **Python**

## Installation
1. Clone this repo:  
   `git clone https://github.com/dotnickcore/my-record-list.git`
2. Run the app:
   `py main.py`
